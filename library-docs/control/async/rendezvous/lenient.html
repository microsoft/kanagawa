Rendezvous with a variable number of threads.

template <typename Request, typename Response, auto N, auto FifoDepth = 32>
class rendezvous ยง

Join up to up to N distinct threads into a group. Each thread contributes one request to an array of requests before that thread blocks. A single function processes the combined array of requests to generate an array of responses. One response is sent to each thread and each thread in the group is unblocked.
